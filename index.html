DOM Manipulation

jQuery provides an easy way to select elements from the DOM using selectors.

You can do so by using HTML elements, classes or id's that you used.

You can also refer to the current element by using the word this.

Once you selected and element from the DOM, the fun begins.

Insert elements, remove them, add classes to them, the list is endless!

Go to their API documentation to find out about all the options.

Very easy to read!

AJAX. A concept

AJAX is not a tool, language or technology, but a concept.

This concept combines technologies to enable asynchronous communication with the server.

Little bits of information are being communicated with the server only if needed.

Your website will be quicker and your users will not have to wait for a full page reload each time.

When to use AJAX

AJAX was made famous by Google, they use it for google Suggest, Gmail and Google Maps.

It is used by populair website such as Facebook and Youtube.

Websites with impatient users, expecting high amounts of user specific data.

Dynamic content

Lot of user action

Investigate SEO implications!

For a static website you better look at other options to enhance performance

Think about caching, local storage, offline storage of data instead
